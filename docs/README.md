# Documentation

## Overview

This directory contains all documentation for the synthesizer project, organized by topic.

## Contents

### [Architecture](architecture/)
High-level architectural concepts, design decisions, and development strategy:
- **Voice Architecture** - Understanding the Voice abstraction and component organization
- **Signal Flow** - How audio flows through the system from generation to output
- **Real-time Audio** - Moving from WAV files to real-time audio output
- **Library Philosophy** - When to use libraries vs build from scratch
- **UI Strategy** - GUI approach, thread safety, and requirements
- **Development Roadmap** - Suggested progression for building the synth
- **Career Alignment** - How this project aligns with audio industry jobs

### [C++ Reference](cpp-reference/)
C++ language concepts and patterns encountered while building the synthesizer.

### [DSP Implementation](adsr-implementation-roadmap.md)
Current implementation plans and progress for specific DSP features.

## Documentation Philosophy

Following the project's learning-focused approach:
- **Concise and scannable** - Clear sections, practical examples
- **Focused on "aha moments"** - Key insights and common gotchas
- **No excessive filler** - Get to the point quickly

All documentation files include table of contents for easy navigation.
